# AI4Trade Signals — 从 0 起步的推广计划

一份务实、可照做的计划。**别指望「上线就爆」**——个人产品靠的是持续、窄、真实。
决定成败的是**流量和信任**,不是代码(代码已经做好了)。

---

## 0. 先立三条规矩(保命)

1. **永不承诺收益。** 所有文案统一「模拟机器人 / 仅供参考 / 交易有风险」。这是法律红线。
2. **先让信号有几周正记录,再猛推。** 现在真实战绩是负的——大推只会换来退款和差评。
3. **诚实是你的差异化。** 别人吹「稳赚」,你敢公开真实战绩(含亏损)——这是信任壁垒,也是最安全的定位。

## 1. 定位(一句话)

> 「每个交易日开盘前,给你几只票的明确买卖价位,按你的本金算好仓位——透明记录、不吹稳赚。」

**目标人群(越窄越好,先选一个):**
- 美股散户新手(想有人帮筛、告诉具体价位)
- 上班族散户(没时间盯盘,只想开盘前花 2 分钟看一眼)

## 2. 免费流量渠道(按性价比排序)

| 渠道 | 怎么做 | 节奏 |
|---|---|---|
| **Reddit** (r/stocks, r/Daytrading, r/wallstreetbets 相关) | 别硬广。发「我用 GitHub Actions + AI 做了个开盘前信号 App,附真实战绩(含亏损)」的建造故事贴,评论区放链接 | 每周 1 篇,换子版 |
| **X / Twitter** | 每个交易日发当天的信号截图 + 收盘复盘(对了/错了都发)。真实连载 = 信任 | 每交易日 1 条 |
| **小红书 / TikTok** | 15–30 秒:今天机器人挑了哪几只、给的价位、结果如何。竖屏、字大 | 每周 3–5 条 |
| **YouTube Shorts** | 同上,英文受众 | 每周 2–3 条 |
| **Product Hunt** | 攒够素材后做一次正式发布(见下) | 一次性 |
| **相关 Discord / 论坛** | 潜水一段时间、真诚参与后再提你的东西 | 持续 |

**核心打法:公开连载。** 每天发「今天信号 → 收盘结果」,把它当成一个真人实验的连续剧。
真实(包括认错)比任何广告都有说服力,也最合规。

## 3. 转化路径

社媒/文章 → 落地页(讲清楚是什么)→ 免费试用一天 → 订阅。
- 落地页文案见 `landing-copy.md`。
- 网站入口:AggreAPI 首页顶部横幅已加(部署后生效)。

## 4. 前 4 周清单

**第 1 周——先备弹药,不急推**
- [ ] 让机器人每个交易日照常跑,积累真实战绩。
- [ ] 建 X / 小红书 / TikTok 账号,统一头像和简介(带「模拟/教育」字样)。
- [ ] 落地页文案就位;把「透明战绩」页数据攒起来。

**第 2 周——开始公开连载**
- [ ] 每交易日发「今日信号」+ 前一日复盘。
- [ ] Reddit 发第 1 篇建造故事贴。

**第 3 周——放大**
- [ ] 挑表现好的内容形式加量。
- [ ] 收集早期用户反馈(App 里已有反馈页),把采纳的更新发出来讲。

**第 4 周——正式发布(可选,等有一定正记录再做)**
- [ ] Product Hunt 发布(文案我可以帮你写)。
- [ ] 同步发 Reddit / X / 邮件列表。

## 5. 该盯的数字

- 访问 → 试用转化率
- 试用 → 付费转化率
- 退订率(高说明信号没价值,别急着买量)
- **单看一个:一周内有几个真实付费用户。** 先做到「10 个愿意付费的人」,再谈规模。

## 6. 花钱投广告?——先别

在你没验证「自然流量能转化 + 信号有正记录」之前,投广告 = 烧钱买亏损。
先用免费渠道跑通转化,再考虑小额测试。
