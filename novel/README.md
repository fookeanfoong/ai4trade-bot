# 《我这修仙，纯属误会》

一部修仙搞笑爽文。主角**何三斤**是土生土长的本地人（不是穿越者），从青云宗药园扫地杂役起步，靠"钝、懒、饿、怂"四大绝技，一路误打误撞捅到修仙界巅峰。

> **一句话钩子**：别人拼死拼活求长生，他只想睡个好觉、吃口热饭——结果一路睡到了大乘期。

**核心笑点公式**：`主角的蠢/懒/怂 + 旁人自作聪明的过度解读 = 神迹`。
